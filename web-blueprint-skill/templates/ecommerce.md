# Plantilla: E-commerce / Tienda Online

## Estructura de Secciones

```
1. Hero — Producto estrella + CTA "Comprar ahora"
2. Categorías — Grid de categorías con imágenes
3. Productos Destacados — Carrusel de bestsellers
4. Beneficios — Envío gratis, pagos seguros, garantía
5. Colecciones — Curated collections por temática
6. Testimonios — Reviews de clientes con fotos
7. Newsletter — Captura de email con descuento
8. Instagram Shop — Feed shoppable
9. Footer — Políticas, envíos, devoluciones, contacto
```

## Copy Sugerido

### Hero
```
Headline: [Producto] que cambia
         tu [beneficio emocional].

Subtítulo: Envío gratis en pedidos +$X. Pago seguro.
Garantía de satisfacción. [N] clientes felices.

CTA: Comprar ahora →
```

### Beneficios típicos (4 íconos)
1. **Envío Gratis** — En pedidos +$X
2. **Pago Seguro** — Encriptación SSL
3. **Garantía 30 días** — Devolución sin preguntas
4. **Soporte 24/7** — WhatsApp directo

## Colores Típico
- **Moda**: Negro + blanco + acento dorado
- **Tecnología**: Azul + cyan + negro
- **Hogar**: Verde + beige + marrón
- **Belleza**: Rosa + dorado + blanco
- **Deporte**: Naranja + negro + blanco

## CTAs
- "Comprar ahora"
- "Añadir al carrito"
- "Ver colección"
- "Suscríbete y ahorra 10%"

## Elementos Únicos
- **Búsqueda con autocomplete**
- **Filtros laterales** (categoría, precio, talla)
- **Quick view** de producto (modal)
- **Carrusel de productos relacionados**
- **Contador de stock** ("¡Solo 3 left!")
- **Reviews con estrellas** + fotos
- **Wishlist** (lista de deseos)
- **Checkout en una página**

## Integraciones Recomendadas
- **Stripe / Mercado Pago** — Pagos
- **Shopify / Medusa** — Backend e-commerce
- **Shippo / Envíame** — Logística
- **Klaviyo** — Email marketing
- **Gorgias** — Soporte al cliente

## Performance Crítico
- LCP < 2s (imagen de producto optimizada)
- Lazy load de imágenes de producto
- CDN para assets estáticos
- Cache de catálogo
- Búsqueda server-side
