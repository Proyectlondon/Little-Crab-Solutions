# Plantilla: SaaS / Software B2B

## Estructura de Secciones

```
1. Hero — Headline + demo video + CTA "Prueba gratis"
2. Logos sociales — "Confían en nosotros"
3. Problema/Solución — Dolor → Cómo lo resuelves
4. Features — 6-8 features con íconos y screenshots
5. Cómo funciona — 3-4 pasos visuales
6. Pricing — 3 planes (Free, Pro, Enterprise)
7. Testimonios — 3 casos con métricas (ej: "+47% en ventas")
8. FAQ — 6-8 preguntas técnicas
9. CTA final — "Empieza gratis hoy"
10. Footer — Docs, API, status, blog
```

## Copy Sugerido

### Hero
```
Headline: [Verbo] tu [sustantivo] con IA.

Subtítulo: La plataforma que [beneficio principal].
Sin [dolor común]. Sin [objeción]. Resultados en [tiempo].

CTA: Empieza gratis → (sin tarjeta de crédito)
```

### Features típicas
1. **Automatización** — Tareas repetitivas al automático
2. **Analítica** — Dashboards en tiempo real
3. **Integraciones** — Conecta con tu stack actual
4. **Seguridad** — SOC2, encriptación, SSO
5. **Escalabilidad** — Crece contigo sin límites
6. **Soporte** — Chat, docs, comunidad

## CTAs
- "Empieza gratis"
- "Agenda demo"

## Elementos Únicos
- **Demo video** en hero (loop, mute, autoplay)
- **Interactive product tour**
- **Pricing calculator** (slider por usuarios/features)
- **Status page** link en footer
- **Changelog** de últimas features
- **Comparison table** vs competidores
- **ROI calculator**

## Performance Crítico
- LCP < 2s (hero image/video optimizado)
- CLS < 0.1 (reservar espacio para imágenes)
- Lazy load de screenshots
- Video en hero: poster image + lazy load

## Integraciones Recomendadas
- **Stripe** — Billing
- **Auth0 / Clerk** — Autenticación
- **Intercom** — Soporte in-app
- **Segment** — Analytics
- **Sentry** — Error tracking
