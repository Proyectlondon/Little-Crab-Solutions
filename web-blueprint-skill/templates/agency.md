# Plantilla: Agencia de Marketing/Diseño

## Estructura de Secciones

```
1. Hero — Headline impactante + showreel
2. Manifiesto — Filosofía de trabajo
3. Servicios — 4-6 servicios principales
4. Portafolio — Grid de proyectos con hover
5. Proceso — Cómo trabajan (brief → concepto → producción → entrega)
6. Equipo — Fotos + roles
7. Casos de Éxito — 3 proyectos con métricas
8. Testimonios — Carrusel de clientes
9. Pricing — 3 paquetes (starter, pro, enterprise)
10. Contacto — Brief form + Calendly
11. Footer — Links, redes sociales
```

### Servicios típicos
1. **Branding & Identidad** — Logo, sistema visual, guidelines
2. **Diseño Web** — Sitios que convierten, no solo se ven bonitos
3. **Social Media** — Contenido que genera comunidad
4. **Producción Audiovisual** — Video, fotografía, motion graphics
5. **Estrategia Digital** — SEO, Ads, analítica
6. **Packaging** — Producto que se vende desde la estantería
